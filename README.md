# Customer_behaviour_Analysis
Data Analysis Project showing customer behaviour analysis using sql, python power bi 

# Project Overview
This project demonstrates an end-to-end data analytics workflow, starting from raw data to business insights. 
It includes data loading using Python, exploratory data analysis (EDA), data cleaning, SQL-based analysis, dashboard creation in Power BI, and final reporting with a presentation.
The goal is to showcase practical skills in data analysis, visualization, and decision support.

# Dataset
Source: CSV / Excel / Database
Description: The dataset contains structured data used for analysis and visualization.
Data includes relevant attributes such as customer details, transactions, metrics, or performance indicators (depending on the use case).
